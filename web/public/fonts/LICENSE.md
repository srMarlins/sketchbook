- `inter-latin-variable-wght.woff2` — **Inter Variable** by Rasmus Andersson — [SIL Open Font License 1.1](https://scripts.sil.org/OFL).

License copies are bundled with each font's source repo on Fontsource. Re-download from `https://cdn.jsdelivr.net/npm/@fontsource/<name>/files/...` to refresh.
